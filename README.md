# SeaGap Docker GUI
This repo contains a GUI for simplifying the usage of F. Tomita's GNSS/A softaware : SeaGap ! This software, writen in Julia allows to use all the state of the art GNSS/A calculation methods and has been explained in Tomita's 2024 article. 
- SeaGap can be found here -> https://github.com/f-tommy/SeaGap
- SeaGap's documenation can be found here -> https://f-tommy.github.io/SeaGapDocs/
- Tomita's 2024 article can be found here -> https://earth-planets-space.springeropen.com/articles/10.1186/s40623-024-01987-9
